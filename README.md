🚀 Portfolio Website (React + Cursor AI)

This is my personal developer portfolio built using React and assisted by Cursor AI. It showcases my skills, projects, and journey as a full-stack developer in a modern, responsive UI.

✨ Features
⚡ Modern and responsive design
🎯 Clean and smooth navigation
🧑‍💻 About, Skills, Projects, and Contact sections
📱 Mobile-friendly layout
🎨 Minimal and professional UI

🛠️ Tech Stack
React.js
JavaScript (ES6+)
HTML5
CSS3
Cursor AI (AI-assisted development)
📁 Project Structure

src/
├── components/
├── pages/
├── assets/
├── App.js
└── index.js

🚀 Getting Started

Clone the repository:
git clone https://github.com/raj11mm/Portfolio-Website-Built-with-React-Cursor-AI-.git

Install dependencies:
npm install

Run the project:
npm start
